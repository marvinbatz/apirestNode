### Features

- ApiRest desarrollado en NodeJS.
- Consumo de Api con JQuery
- Maquetación Bootstrap 5

#### Nodemon

Para inicializar el servidor se utilizó nodemon en el puerto 3000:
```javascript
  "scripts": {
    "start": "nodemon src/app.js"
  },
  
// setings
app.set('port', process.env.PORT || 3000);


```
